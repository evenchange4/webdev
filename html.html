大腸包小腸

  • HTML 標籤可以互相嵌套 (nesting)
  • 但不能隨便包

Nesting Rule

  • 原則上 inline element 不能包 block element
  • <p>、<h1>~<h6> 裡頭只能包 inline element
  • <strong> 裡頭不能包 <div>
  • <ul>、<ol> 裡面只能包 <li>
  • <tr> 裡面只能包 <td>，外頭必須要有 <table>
  • 但是……
